# software-requirements-specification
Название проекта: Chatty 2.0  Тип проекта: Веб-приложение для изучения английского языка.  Основная цель: Расширить функциональность предыдущей версии, добавив инструменты для работы со словарём и карточками для изучения слов.
Ключевые возможности (основные функции):

Работа со словарём:
Поиск перевода и подробной информации о словах.
Прослушивание произношения.
Просмотр контекстных видео с использованием слова.
Ведение личного словаря пользователя (добавление, просмотр, удаление слов).

Работа с карточками:
Создание карточек для запоминания слов из личного словаря.
Управление карточками (добавление, удаление).
Режим самопроверки знаний (пользователь смотрит на слово и проверяет, помнит ли он перевод).

Пользователи:
Ученики (изучают английский, выполняют задания).
Преподаватели (проводят уроки, составляют задания).

Основные требования к системе:
Работа через современные браузеры (Chrome, Safari).
Интеграция со сторонними подключаемыми словарями и библиотеками видео/аудио.
Высокая производительность (загрузка страниц до 3 секунд, добавление слов до 1 секунды).
Доступность 99% времени в дневные часы.
Обязательная авторизация пользователей.
Использование PostgreSQL и стандарта HTML 5.0.
